# GreatIdea.dev

GreatIdea.dev is a team of software engineers from Mississippi dedicated to serving the community by providing high-quality web solutions and robust applications to businesses and individuals looking to jumpstart their online presence and embrace the digital world.

## Getting started

To get started with this template, first install the npm dependencies:

```bash
npm install
```

Next, run the development server:

```bash
npm run dev
```

Finally, open [http://localhost:3000](http://localhost:3000) in your browser to view the website.
